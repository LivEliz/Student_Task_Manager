# Student Task Manager

A simple task manager web app to help students organize and track their daily tasks and assignments.

Description
The Student Task Manager is a project built to allow users to create, view, update, and delete tasks. It provides a basic yet useful interface for keeping track of to-dos and managing them quickly. This project was built as part of hands-on learning to practice web development skills.

Features
- Add new tasks
- View existing tasks
- Edit task details
- Delete tasks when completed

Built With
- JavaScript
- HTML
- CSS
